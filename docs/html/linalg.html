pyvib.linalg.SVD(x, m)¶

Produce the singular value decomposition of a signal Uses the scikit learn.randomized SVD function

Parameters:
  • x (array 1D float) – The signal to decompose
  • m (int) – Number of singular values wanted
Returns:

  • U (array 2D float)
  • sigma (array 1D float) – The singular values ranging from highest to lowest energy
  • V (array 2D float)
